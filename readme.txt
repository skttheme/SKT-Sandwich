== Description ==

SKT Sandwich is an easy to use gutenberg block editor based template can be used for pizza, pizzeria, burger, multi cuisine, Mexican, chinese, fast food, bakery, cafe, coffee, tea shop, restaurant, bistro, joint, pub, bar, fine dine in, takeaway, delivery, recipe, french fries, food blogging, vlog, hotel, lodge, resort, homestay business model or any such hospitality services. Compatible with WooCommerce and also SEO plugins.
